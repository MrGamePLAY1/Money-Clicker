# Money Clicker

A simple and addictive incremental clicker game built with HTML, CSS, and JavaScript. Click the button, earn money, and buy upgrades to automate your income!

## Features

- **Click to Earn:** The core mechanic! Click the "Print Money" button to increase your bank balance.
- **Upgrades:**
    - **Auto Clicker:** Automatically earns money for you every second.
    - **Fast Auto Clicker:** A more powerful upgrade that boosts your passive income even faster.
- **Save System:** Automatically saves your progress locally so you don't lose your hard-earned cash.
    - **Manual Save:** Click the "Save" button to save immediately.
    - **Auto Save:** The game saves automatically in the background.
    - **Load Game:** Restore your progress with the click of a button.
- **Sleek UI:** Dark mode interface with satisfying animations and responsive design.

## How to Play

1.  Open `index.html` in your web browser.
2.  Click the "Print Money" button to start earning.
3.  Once you have enough money, the **Upgrades** section will appear.
4.  Purchase upgrades to increase your CPS (Clicks Per Second).
5.  Watch your bank balance grow!

## Technologies Used

- **HTML5:** For structure and layout.
- **CSS3:** For styling, animations, and responsive design.
- **JavaScript (ES6):** For game logic, state management, and local storage handling.

## Installation

No installation required! Simply clone the repository and open the file:

```bash
git clone https://github.com/MrGamePLAY1/Money-Clicker.git
cd Money-Clicker
# Open index.html in your preferred browser
```
